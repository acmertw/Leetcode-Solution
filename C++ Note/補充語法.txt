1. istringstream
Header : <sstream>

Ex1 :
string s = "   abc def  i jk";
istringstream is(s);
string tmp;
while (is >> tmp) {
    cout << tmp << endl;
}

* output : 
abc
def
i
jk

Ex2 :
is >> tmp  ==> 如果最後有成功則在做if判斷時,is會變成true
string s = "a b";
istringstream is(s);

// Point 1 : current "is" is true

is >> tmp;
// Point 2 : current "is" is true , tmp is "a"

is >> tmp;
// Point 3 : current "is" is true , tmp is "b"

is >> tmp
// Point 4 : current "is" is false
